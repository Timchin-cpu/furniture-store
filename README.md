# Магазин мебели SoFi

Современный адаптивный интернет-магазин мебели, созданный на React.js.

## 🚀 Особенности

- ✨ Современный минималистичный дизайн
- 📱 Полностью адаптивный (мобильные устройства, планшеты, десктоп)
- 🎨 Использование шрифта Manrope из Google Fonts
- 🛒 Функционал корзины покупок
- ❤️ Избранные товары
- ⏱️ Таймер ежедневных предложений
- 🎠 Слайдер на главной странице
- 📧 Форма подписки на рассылку

## 📦 Установка

1. Убедитесь, что у вас установлен Node.js (версия 14 или выше)

2. Установите зависимости:
```bash
npm install
```

## 🎯 Запуск проекта

Для запуска проекта в режиме разработки:

```bash
npm start
```

Приложение откроется по адресу [http://localhost:3000](http://localhost:3000)

## 🏗️ Сборка для production

Для создания оптимизированной сборки:

```bash
npm run build
```

Готовые файлы будут находиться в папке `build/`

## 🎨 Технологии

- **React 18** - библиотека для создания пользовательских интерфейсов
- **React Hooks** - useState, useEffect для управления состоянием
- **CSS3** - современные стили с Flexbox и Grid
- **Google Fonts** - шрифт Manrope
- **Unsplash API** - изображения товаров

## 📱 Адаптивность

Сайт полностью адаптирован для:
- 📱 Мобильных устройств (до 640px)
- 📱 Планшетов (641px - 1024px)
- 💻 Десктопов (1025px и выше)

## 🎯 Функционал

### Реализованные возможности:
- Навигационное меню с мобильной версией
- Слайдер товаров на главной странице
- Категории товаров
- Каталог с товарами
- Добавление в корзину
- Добавление в избранное
- Таймер акций
- Форма подписки на рассылку
- Footer с контактами и ссылками

### Примечание:
Это фронтенд-проект с фейковыми данными для демонстрации в резюме. Все данные статичные, без подключения к серверу.
